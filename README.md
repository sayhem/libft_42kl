# Libft
Libft is a custom implementation of the standard C library, developed as part of the 42 School curriculum. The goal of this project is to recreate commonly used functions from `<string.h>`, `<ctype.h>`, `<stdlib.h>`, and other standard libraries.

The library is divided into two main parts:

### Part 1 – Standard Library Functions
Reimplementation of functions such as:
- `ft_strlen`
- `ft_strcpy`
- `ft_strncmp`
- `ft_strdup`
- `ft_memset`
- `ft_memcpy`
- `ft_memmove`
- `ft_isalpha`, `ft_isdigit`, `ft_isalnum`, etc.
- `ft_atoi`

### Part 2 – Additional Functions
Custom utility functions including:
- String manipulation utilities
- Memory allocation helpers
- Linked list functions
